# ChatGPT Clone Workspace

This is a simple web-based ChatGPT clone built with Node.js, Express, and vanilla HTML/CSS/JS.

## Features
- Clean, responsive UI inspired by ChatGPT dark mode
- Real-time interaction with the OpenAI API via backend proxy
- Maintains conversation history within the browser session

## Prerequisites
- Node.js installed on your machine
- An OpenAI API Key from [platform.openai.com](https://platform.openai.com/)

## Setup Instructions

1. **Open your terminal** and navigate to this folder:
   ```cmd
   cd c:\Users\hp\Desktop\Mukesh\chatgpt-clone
   ```

2. **Install Dependencies**
   Run the following command:
   ```cmd
   npm install
   ```

3. **Configure Environment Variables**
   Open the `.env` file and replace `your_openai_api_key_here` with your actual OpenAI API key (it should start with `sk-...`).

4. **Start the Server**
   Run the following command to start the backend server:
   ```cmd
   npm start
   ```

5. **Access the App**
   Open your web browser and navigate to:
   http://localhost:3000
